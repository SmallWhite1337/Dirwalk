Программа dirwalk выполняет рекурсивный обход файловой системы,
начиная с указанного каталога, и выводит пути к найденным объектам.
Поддерживается фильтрация по типам файлов и сортировка результатов.

Сборка
Для сборки программы используйте Makefile:
make
Сборка отладочной версии:
make debug
Очистка файлов сборки:
make clean
make distclean

Запуск программы
./dirwalk [каталог] [опции]
Если каталог не указан, используется текущий каталог.

Опции
-f    выводить только обычные файлы
-d    выводить только каталоги
-l    выводить только символические ссылки
-s    сортировать вывод в лексикографическом порядке

Примеры
Обход текущего каталога:
./dirwalk
Обход каталога /etc:
./dirwalk /etc
Только обычные файлы:
./dirwalk /etc -f
С сортировкой:
./dirwalk /etc -s
